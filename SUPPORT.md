# LokalBot Support

Start with the [README](README.md) and [release guide](RELEASING.md). For a bug,
question, or feature request, open a GitHub issue:

https://github.com/stevyhacker/lokalbot/issues/new/choose

Include the LokalBot version, macOS version, Mac model, the exact action that
failed, and the smallest reproducible steps. You may attach a redacted excerpt
from LokalBot's debug log, but never post meeting audio, transcripts, API keys,
tokens, model-server credentials, or other private material.

For a suspected security vulnerability, do not open a public issue. Follow
[SECURITY.md](SECURITY.md) instead.

This is a community open-source project; response times are not guaranteed.
