Help and feedback

Support

LokalBot is an open-source community project. The issue tracker is the public support desk.

Before opening an issue

Check the README, install the latest release, and retry the smallest failing flow.

Report a bug or request a feature

Open a GitHub issue

Include the LokalBot version, macOS version, Mac model, exact action that failed, and reproducible steps. A redacted debug-log excerpt can help.

Never post meeting audio, transcripts, screenshots, API keys, tokens, model-server credentials, or other private material.

Security reports

Do not disclose vulnerabilities in a public issue. Use GitHub's private vulnerability-reporting form. If it is unavailable, open a public issue with no exploit details and request a private contact channel.

Response expectations

Support is best-effort and response times are not guaranteed. Clear reproduction steps and a privacy-safe log excerpt make a response much more likely.
