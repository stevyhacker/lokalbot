Privacy Policy

LokalBot is local-first. It has no LokalBot account, analytics service, advertising SDK, telemetry backend, or hosted AI service.

What stays on your Mac

The project does not receive your recordings, transcripts, summaries, screenshots, prompts, files, calendar events, or usage history. LokalBot can store these in its Application Support directory:

  • meeting microphone and system-audio tracks;
  • transcripts, summaries, notes, search indexes, and meeting metadata;
  • downloaded speech, embedding, and language models;
  • opt-in app/window activity, screenshots, and locally extracted text;
  • opt-in Agent Mode sessions and runtime files; and
  • preferences, diagnostic logs, and encryption keys.

Screenshots are off by default, encrypted at rest, and deleted after 14 days by default. Their extracted text follows the same retention unless you explicitly keep it. Dictation scratch audio is deleted after transcription by default.

When LokalBot uses the network

  • Models: model metadata and files from Hugging Face or a publisher's download host. Selected first-use models can download automatically.
  • Updates: the public GitHub Releases appcast and a signed update when you check or enable automatic checks.
  • Optional remote inference: an Ollama or OpenAI-compatible URL you configure. Loopback stays on your Mac. A non-loopback origin must be explicitly approved before it receives meeting, workday, or agent context.
  • Optional Agent Mode: setup downloads its pinned runtime. Commands you approve can access files or the network with your macOS user permissions.

Those services receive ordinary connection metadata such as IP address and request headers. LokalBot adds no advertising identifier and does not use those requests for tracking. A remote server's operator controls its own privacy terms.

Permissions and consent

Microphone and system-audio permissions enable recording. Calendar access helps detect and title meetings. Accessibility enables browser-meeting detection, Cotyping, dictation insertion, and approved agent interaction. Screen Recording enables opt-in screen context.

Recording defaults to manual on a fresh install. You are responsible for obtaining any consent required before recording other people.

External-agent access

The bundled CLI and MCP interface are read-only and refuse library access until you enable Agent Access under Settings → Privacy. An enabled external tool runs as your macOS user, so connect only tools you trust.

Deletion, security, and changes

You can delete meetings in the app or remove LokalBot's Application Support directory. No software can promise absolute security. Report vulnerabilities through the private process in the security policy. Material policy changes will update this date and appear in release notes.

Contact

Use the support page for privacy questions. Never include recordings, transcripts, secrets, or other sensitive data in a public issue.
